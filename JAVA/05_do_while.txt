- quiz4 - 3자리의 양의 정숫값(100~999)을 읽는 프로그램을 작성하라
         (입력한 값이 3자리 양의 정숫값이 아니면 다시 입력하게 할 것).
-----------------------------------------------------------------------------
    import java.util.Scanner;
    Scanner sc = new Scanner(system.in);
    int a;
   
    do {
            System.out.print("세 자리의 정숫값 : ");
            a = sc.nextInt();
            }while(a < 100 || a >999);{		
            System.out.println("입력한 값은" + a + "입니다");
    }
------------------------------------------------------------------------------
입력값 : 1234
세자리의 정숫값 : 1234
입력값 : 123
입력한 값은 123 입니다.

- quiz 8 - 입력한 값의 개수만큰 ‘*’와 ‘+’를 번갈아 가며 표시하는 프로그램을 작성하자 (마지막에는 줄 바꿈 문자를 출력)
-----------------------------------------------------------------------------------------------------------------
Scanner sc = new Scanner(System.in);
System.out.print("몇개를 표시할까요 : ");
int no = sc.nextInt();
    while(no>0) {
        if(no%2==0) {

            System.out.print("+");
            no--;
        }
        if(no%2==1) {

            System.out.print("*");
            no--;
        }
    }
--------------------------------------------------------------------------------------------------------------
출력값 :
몇개를 표시할까요 : 9
